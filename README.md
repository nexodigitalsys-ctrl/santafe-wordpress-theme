# Santa Fe Tailwind Theme

Tema WordPress da Santa Fe Tailwind Theme.

Preview: https://santafe.nexo-digital.app/es/

Pasta de instalacao esperada na Hostinger:

```text
public_html/wp-content/themes/santafe_wp_theme
```

O tema mantem Tailwind via CDN por enquanto.

O deploy Git deve apontar para:

```text
public_html/wp-content/themes/santafe_wp_theme
```
